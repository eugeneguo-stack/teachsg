# Coordinate Geometry
[stextbox caption="Cartesian coordinate" stextbox id="default"]$(x,y)$
[/stextbox]

[stextbox caption="Gradient" stextbox id="download"]$m=\dfrac{y_1-y_2}{x_1-x_2}$
[/stextbox]

[stextbox caption="Equation" stextbox id="info"]$y-y_1=m(x-x_1)$
$y=mx+c$
*Vertical line: $x=a$
*Horizontal line: $y=b$
[/stextbox]

[stextbox caption="Length" stextbox id="default"]$\text{Length } = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$
[/stextbox]
